Sync automatique

Les données sont sauvegardées dans un Gist privé nommé mykajy_data.json sur ton compte GitHub.

La synchronisation utilise un Gist privé sur ton compte GitHub. Aucun abonnement requis.

1
Aller sur GitHub
Settings → Developer settings → Personal access tokens → Tokens (classic)
2
Générer un token
Cliquer Generate new token (classic), donner un nom (ex: "MyKajy"), cocher uniquement le scope gist, puis générer.
3
Coller le token ci-dessous
Le token commence par ghp_
